## 💡 Key Things I Learned Today

1. Linux uses three standard I/O streams: `stdin`, `stdout` and `stderr`.
2. Output, input and errors can be redirected independently.
3. `/dev/null` is useful for discarding unwanted output.
4. `diff`, `cmp` and `sdiff` compare files in different ways.
5. Environment variables store configuration values used by the shell and applications.
6. Linux provides commands like `ps`, `top`, `jobs`, `bg` and `fg` for process management.
7. Signals such as `SIGTERM` and `SIGKILL` are used to terminate processes.
8. Exit status (`$?`) is essential for writing reliable Bash scripts.
